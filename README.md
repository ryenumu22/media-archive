# media-archive

A personal archive for storing images and videos.

## Structure

Organize your media however works best for you. Some ideas:

```
media-archive/
├── photos/
│   ├── 2024/
│   └── 2025/
├── videos/
│   ├── travel/
│   └── events/
└── screenshots/
```

## Notes

- Large files (videos especially) may benefit from [Git LFS](https://git-lfs.github.com/) to avoid bloating the repo history.
- To enable Git LFS: `git lfs install` then `git lfs track "*.mp4" "*.mov" "*.avi"`
